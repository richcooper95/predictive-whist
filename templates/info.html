What's What's Trumps?

What's Trumps? is a website for scoring games of Nomination Whist.
It provides way to score and track the games you play, so no one argues about points, and the trump suit is veeeery obvious each round to save the inevitable, repeated calls of: "What's trumps?"

What's Nomination Whist?

Nomination Whist (n.) - a card game in which players try to predict the number of tricks they will win each round, and inevitably get frustrated when they don't.
See also: Oh Hell!, Oh Shit! (if you're really frustrated), Oh Well! (for those with some perspective), Oh Pshaw!, Elevator (l'Ascenseur in France), Contract Whist, Predictive Whist (possibly only in my family), Boerenbridge, German Bridge, Kachuful, Up and Down the River, Bust, Blob, Blackout.
For a full explanation of the how to play, see the Rules section.
